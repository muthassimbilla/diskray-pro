# Product Requirement Document (PRD): PC Explorer & Manager (Tauri Edition)

## ১. প্রজেক্টের সংক্ষিপ্ত বিবরণ (Project Overview)
এই টুলটি একটি হাই-পারফরম্যান্স ডিস্ক স্পেস অ্যানালাইজার যা Rust-এর গতি এবং React-এর আধুনিক ইন্টারফেস ব্যবহার করে তৈরি। এটি উইন্ডোজ এবং লিনাক্স (Fedora) উভয় প্ল্যাটফর্মে ফাইল সিস্টেম স্ক্যান করে বড় ফাইলগুলো শনাক্ত করবে এবং ক্যাটাগরি অনুযায়ী ফিল্টার করার সুবিধা দেবে।

---

## ২. টেকনিক্যাল স্ট্যাক (Technical Stack)
- **Backend:** Rust (Tauri v2) - ফাইল সিস্টেম অ্যাক্সেস এবং গতির জন্য।
- **Frontend:** React + TypeScript - ইউজার ইন্টারফেসের জন্য।
- **Libraries:**
    - `walkdir`: দ্রুত ফাইল স্ক্যানিংয়ের জন্য।
    - `serde`: ডাটা সিরিয়ালাইজেশনের জন্য।
    - `tauri_plugin_opener`: সিস্টেম ফাইল হ্যান্ডলিংয়ের জন্য।
- **Styling:** Tailwind CSS / Modern CSS (প্রফেশনাল লুকের জন্য)।

---

## ৩. মূল ফিচারসমূহ (Core Features)

### ৩.১ অ্যাডভান্সড স্ক্যানিং (Scanning)
- **Custom Path:** ব্যবহারকারী যেকোনো নির্দিষ্ট ফোল্ডার বা সম্পূর্ণ ড্রাইভ স্ক্যান করতে পারবেন।
- **Inaccessible Skip:** পারমিশন নেই এমন সিস্টেম ফোল্ডারগুলো স্ক্যান করার সময় অটোমেটিক স্কিপ হবে (ক্রাশ রোধ করতে)।
- **Deep Scan Logic:** Rust-এর মাধ্যমে অত্যন্ত দ্রুতগতিতে মেমোরি রিড করা।

### ৩.২ স্মার্ট ফিল্টারিং (Filtering)
ব্যবহারকারী নিচের ক্যাটাগরি অনুযায়ী ফাইল ফিল্টার করতে পারবেন:
- **All Files:** সাইজ অনুযায়ী সব ফাইল।
- **Images:** .jpg, .png, .webp, .gif ইত্যাদি।
- **Archives:** .zip, .rar, .7z, .tar.gz (স্টোরেজ ক্লিনিংয়ের জন্য জরুরি)।
- **Documents:** .pdf, .docx, .txt, .xlsx।

### ৩.৩ ডেটা অ্যানালাইসিস (Data Analysis)
- **Top 100 Files:** স্ক্যান করা পাথের মধ্যে সবচেয়ে বড় ১০০টি ফাইলের তালিকা।
- **Human Readable Size:** ফাইল সাইজকে অটোমেটিক KB, MB বা GB-তে কনভার্ট করে দেখানো (যেমন: 1.5 GB)।

## ৪. ইউজার ইন্টারফেস ডিজাইন (UI/UX Design)

### ৪.১ লেআউট (Layout)
- **Dashboard Header:** বর্তমান স্ক্যানিং পাথ এবং একটি 'Start Scan' বাটন।
- **Sidebar/Control Panel:** ক্যাটাগরি সিলেক্টর (All, Images, etc.) এবং সেটিংস।
- **Main Table:** প্রফেশনাল ডাটা গ্রিড যেখানে থাকবে:
    - **Name:** ফাইলের নাম (আইকনসহ)।
    - **Extension:** ফাইলের ধরন।
    - **Size:** ফাইলের আকার (Formatted)।
    - **Path:** ফাইলের পূর্ণ অবস্থান।

### ৪.২ ইউজার এক্সপেরিয়েন্স (UX)
- **Dark Mode Support:** সিস্টেম থিমের সাথে অটোমেটিক অ্যাডজাস্ট।
- **Loading State:** স্ক্যান করার সময় একটি এনিমেটেড স্পিনার বা প্রগ্রেস টেক্সট দেখানো।
- **Interactive Rows:** কোনো ফাইলের ওপর হোভার করলে সেটি হাইলাইট হবে।

---

## ৫. রোডম্যাপ (Development Roadmap)

- **Phase 1 (Setup):** Tauri v2 এবং Rust এনভায়রনমেন্ট সেটআপ (সম্পন্ন)।
- **Phase 2 (Logic):** Rust ব্যাকএন্ডে `walkdir` এবং ফিল্টারিং লজিক ইমপ্লিমেন্ট করা।
- **Phase 3 (Frontend):** React ড্যাশবোর্ড এবং টেবিল ভিউ তৈরি করা।
- **Phase 4 (Bridge):** `invoke` কমান্ডের মাধ্যমে ফ্রন্টএন্ড এবং ব্যাকএন্ড কানেক্ট করা।
- **Phase 5 (Refinement):** আইকন যোগ করা এবং পারফরম্যান্স অপ্টিমাইজেশন।
